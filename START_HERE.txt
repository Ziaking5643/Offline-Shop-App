â“ COMMON QUESTIONS

Q: How do I stop the application?
A: Click on the black command window and press Ctrl+C, then press Y

Q: How do I restart?
A: Simply run run.bat again

Q: Where is my data stored?
A: In db.sqlite3 file - backup this file regularly!

Q: Can I use this on multiple computers?
A: Copy the entire shop_app folder and run on another computer

Q: What if I forgot the admin password?
A: Delete db.sqlite3 and run run.bat again (this resets everything)

Q: Can I add more users?
A: Yes, through the Admin panel
